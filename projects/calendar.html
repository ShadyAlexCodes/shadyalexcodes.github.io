Holiday Calendar

The concept

When the Holiday's come around families across the world are often purchasing Holiday Advent Calendars.
This project brings the calendar to you, and introduces holiday activities for all ages! From simple
coding questions all the way to facts about computer science.
(Click the christmas tree!)



The Building Process

The inspiration for this idea came from a neighbor who had just bought one for themselves,
and during that time I was looking at other projects such as this one.
After much research, and feedback on the idea I began developing it.

The development of this project began with dots!
I wanted to create something cute, and fun to represent my questions.

After arranging the dots, I numbered them, and
began experimenting with different concepts and things to display these questions.
While browsing Codepen I found the project Lightbox.
With this project I was able to create a button click that told me the question.

With the HTML and CSS part out of the way, I needed someway
for the users to respond to these questions.
I continued by creating a Javascript answering system.

The Javascript
function answerClicked() {
var answer = document.getElementById('questionAnswer').value.toLowerCase();
var correctAnswers = ['vint cerf', 'robert e. kahn', 'robert kahn', 'print.ln("hello world!");',
'#003366', '003366', 'lossless', '1995', 'bill gates', 'september 4, 1998', '9/4/1998', '2004', ];
if(correctAnswers.includes(answer)) {
swal("Great job!", "You got the question correct! Click 'Tis the season to get to the main page!", "success");
} else {
swal("Awww man!", "You got the question wrong!", "error");
} };
When the user responds to a question it'll run function `answerClicked();`
This function ran through a pre-created array list to see if the user
response was equal to one of the ones in the list. Then responded
to the user telling them if they were right or wrong.
